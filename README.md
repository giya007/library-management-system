# Library Management System (Python)

A simple terminal-based Library Management System developed using Python as part of a first-year BTech group mini project.

## Overview
This project is a menu-driven application that performs basic library operations using text files for data storage. It was created to practice Python programming fundamentals and understand file handling concepts.

The system runs in the terminal and manages available and issued books through simple user inputs.

## Features
- View available and issued books
- Add new books to the library
- Issue books using Book ID
- Return issued books
- Persistent storage using text files

## How It Works
- `available.txt` stores available books
- `issued.txt` stores issued books
- Temporary files are used while issuing and returning books to update records safely
